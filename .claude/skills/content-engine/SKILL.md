# Content Engine

Generate platform-native content for careers.ky from source material (job postings, features, announcements, milestones).

## Process

### 1. Extract Core Ideas
- Identify the key message, value proposition, and target audience
- Pull out quotable facts, statistics, or unique angles
- Determine the emotional hook (opportunity, urgency, community, pride)

### 2. Platform Variants

**X (Twitter)**
- 280 char limit, punchy and conversational
- Lead with hook or question
- Hashtags: #CaymanJobs #CareersKY #CaymanIslands
- Thread format for complex topics (3-5 tweets max)

**LinkedIn**
- Professional tone, 150-300 words
- Open with insight or data point
- Structure: hook → context → value → CTA
- Tag relevant companies/people when appropriate

**TikTok / Reels Script**
- 30-60 second script with visual cues
- Hook in first 3 seconds
- Conversational, authentic tone
- End with CTA or question to drive comments

**Newsletter Blurb**
- 100-200 words, informative tone
- Context for the Cayman job market audience
- Link to full content on careers.ky

**WhatsApp Broadcast**
- Under 150 words, casual and direct
- Key info upfront (what, where, how)
- Easy to forward/share

### 3. Adaptation Guidelines
- **Cayman context**: Reference local relevance (work permits, local hiring, island life)
- **Brand voice**: Helpful, modern, no corporate jargon
- **CTA**: Always drive to careers.ky — Express Interest for jobs, sign-up for platform features
- **Timing**: Note optimal posting times if relevant (Cayman is EST/UTC-5)

### 4. Output Format
Present each platform variant with:
- Platform name and format
- Character/word count
- The content itself
- Any suggested media (image, video, carousel)
