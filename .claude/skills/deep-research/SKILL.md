# Deep Research

Structured research workflow for careers.ky topics — market analysis, competitive intelligence, regulatory research, feature validation.

## Process

### 1. Decompose
Break the research question into 3-7 sub-questions that together answer the main question. For each sub-question, identify:
- What type of source would best answer it (web, documentation, data, expert opinion)
- What search terms to use
- What would constitute a good vs. incomplete answer

### 2. Multi-Source Search
For each sub-question, search across available sources:
- **Web search**: General information, news, blog posts, reports
- **Company/product sites**: Direct competitor analysis, feature comparison
- **Government sources**: Cayman Islands regulations, WORC policies, work permit rules
- **Technical docs**: Framework capabilities, API limitations, integration options

Track sources with URLs for citation.

### 3. Evaluate & Filter
For each finding:
- Assess source credibility (official > news > blog > forum)
- Check recency (prefer sources from last 12 months)
- Cross-reference claims across multiple sources
- Flag conflicting information

### 4. Synthesize
Produce a structured report:

```
## Research: [Topic]

### Key Findings
- [Finding 1] — [Source]
- [Finding 2] — [Source]

### Analysis
[Synthesis of findings, patterns, implications]

### Recommendations
[Actionable next steps for careers.ky]

### Sources
1. [Title](URL) — [Brief description of relevance]
```

### 5. Identify Gaps
Note what couldn't be answered and suggest:
- Additional research needed
- People to talk to
- Data to collect

## Cayman-Specific Research Areas
- Work permit regulations and LMIA equivalents
- Local employer landscape and hiring patterns
- Competitor job platforms (regional and local)
- Government employment initiatives
- Immigration policy changes affecting hiring
